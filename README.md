# 🧠 Assistente de Currículo e Vagas de Emprego

## 📌 Sobre o projeto
Este projeto é um assistente virtual com Inteligência Artificial que ajuda pessoas a melhorarem seus currículos, entenderem vagas de emprego e se prepararem para entrevistas.

## 🎯 Objetivo
Ajudar usuários a:
- Criar e melhorar currículos
- Entender requisitos de vagas
- Receber dicas de carreira
- Se preparar para entrevistas

## 👥 Público-alvo
Pessoas em busca do primeiro emprego, recolocação profissional ou orientação de carreira.

## ⚙️ Como funciona
O assistente utiliza uma base de conhecimento simples e responde perguntas de forma objetiva, evitando inventar informações.

## 📁 Estrutura do projeto
- data/ → base de conhecimento
- docs/ → documentação e prompts
- src/ → código da aplicação

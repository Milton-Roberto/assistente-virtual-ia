# Pitch do Projeto

Criei um assistente virtual para ajudar pessoas a melhorarem seus currículos e se prepararem para vagas de emprego.

O sistema utiliza uma base de conhecimento simples para garantir respostas confiáveis e evitar informações inventadas.

O objetivo é apoiar principalmente pessoas em início de carreira ou em transição profissional, ajudando-as a se posicionarem melhor no mercado de trabalho.
